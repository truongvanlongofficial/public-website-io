Privacy Policy

Last updated: 2026-06-30

Overview

Audio Metadata Editor helps users view and edit metadata for local audio files they choose on their Android device.

Files And Data You Select

The app works with audio files selected by the user through the Android file picker. It reads available metadata such as title, artist, album, genre, year, track number, file name, MIME type, and whether cover art is present.

The app does not scan the device in the background and does not request broad media library access for this workflow.

Processing And Storage

Metadata reading and supported MP3 tag writing are performed on the device. When saving is supported, the app creates a new MP3 copy instead of silently changing the original selected file.

Saved MP3 copies are written to the device media library under Music/Audio Metadata Editor when supported by the Android version. The app stores a small local recent-edit history on the device so users can see recently edited file names.

Permissions

The app requests Internet access so third-party ads can load. File access is handled through the Android file picker for files the user selects. The current app manifest does not request contacts, location, camera, microphone, notifications, or background access permissions.

Ads, Analytics, Billing, And Third-Party SDKs

The app includes Google Mobile Ads. Ads may use device, ad, diagnostic, and usage data according to Google's policies and the user's device settings. Interstitial ads may appear after a successful save. Banner ad support may exist in the app configuration, but ads should not block reading or editing a single selected file.

The current implementation does not include account login, subscriptions, in-app purchases, or a custom analytics workflow.

Data Sharing

User-selected audio files are processed on the device by the app workflow. The app does not upload selected audio files to a server. Third-party ad SDKs may process advertising and diagnostics data according to their own privacy policies.

Children

Audio Metadata Editor is intended for a general audience and is not directed to children.

Changes To This Policy

This policy may be updated when app behavior, permissions, ads, storage, or file handling changes.

Contact

For privacy questions, use the developer contact listed on the app's Google Play listing.
